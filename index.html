What is #MAPMYKOCHI?

FISAT Free Software Cell (FFSC), in collaboration with ICEFOSS 2020, is striving to undertake a coordinated mapping event Mapathon, under the tagline #MAPMYKOCHI. The committee is to implement the project from February 3rd along with the commencing of ICEFOSS 2020 till the end of February.

Why #MAPMYKOCHI?

The session strives to improve the present open source map where the participants are invited to make online map improvements in their local area to improve coverage and to help disaster risk assessment and energy management using OpenStreetMap for the same. The main idea behind the "mapathon" is to improve the map data regarding the various locations in this area of Kerala for OpenStreetMap. This is to encourage fairer competition in the map app field against proprietery options like Google Street maps, Waze and the like as well as improve existing open source software as a community. Contribution and cooperation are the strengths that we, the open source community, have against corporate powers. As such, it is our responsibility to help contribute to this large community project.

How to Map?

1) Sign Up for OpenStreetMap

#MapMyKochi is done in Open Street Map. OpenStreetMap is a community-driven project to create a free and editable map of the world. Before you can contribute to #MapMyKochi you will need to create an OpenStreetMap user account, which is what you will use to login to the Tasking Manager, which is a software that help contributions more easier.

Sign Up on Open Street Map!

2) Learn How to Use the Tasking Manager

The Tasking Manager is the tool / website which is used by #MapMyKochi to conduct Mapathon. Its main purpose is to divide up large areas that need mapping into small, easy to map Tasks. It organizes the small Tasks into a larger group called a Project. It provides a user interface to find a Project and then Check-Out and Check-In the small Tasks so you can map them in OpenStreetMap using one of the standard OpenStreetMap editors. Learning how to actually map is shown in the videos below.

3) Start Mapping!

> How to map Buildings?

Building mapping is the most needed and the most common type of mapping which can be done easily. Using the OpenStreetMap web based editor called the iD editor, building mapping is fairly easy. This video will show you the basics of how to map buildings using the iD editor.

> How to map Roads?

Road mapping is the next most common type of mapping does. Physically mapping roads is very easy, but deciding how to classifiy them is the real challenge. This video will give you basics of how to map roads using the iD editor.
